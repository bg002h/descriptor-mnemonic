# v0.4 BIP 388 Modern Post-Segwit Surface — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Lift the encode/decode rejection of three currently-rejected modern post-segwit BIP 388 top-level descriptor types (`wpkh(@0/**)`, `sh(wpkh(@0/**))`, `sh(wsh(...))`) in the `md-codec` crate, completing the modern post-segwit subset of BIP 388 while remaining deliberately narrower than BIP 388.

**Architecture:** Wire-format additive at the type level — no new tags (Pkh=0x02, Sh=0x03, Wpkh=0x04 already allocated in `bytecode/tag.rs`), no new header bits, no schema bump. The `Wsh` subtree handler is split into a body+wrapper layer pattern; `Sh` dispatches via a peek-before-recurse restriction matrix that admits only `Sh→Wpkh` and `Sh→Wsh`. Default path-tier selector is scoped to v0.4-introduced types only (preserves wsh/tr no-origin BIP 84 fallback).

**Tech Stack:** Rust 1.85, miniscript v13 (apoelstra fork via workspace `[patch]`), bitcoin 0.32, bech32 0.11, clap 4, serde, thiserror 2.

**Source spec:** `design/SPEC_v0_4_bip388_modern_segwit_surface.md` (commit `9397c16`).

### Task 2.4: Implement `decode_sh_inner` with peek-before-recurse

- [ ] **Step 1: Add new function**

```rust
fn decode_sh_inner(
    cur: &mut Cursor,
    keys: &[DescriptorPublicKey],
) -> Result<Descriptor<DescriptorPublicKey>, Error> {
    // Three tag families exist; the dispatcher layers MUST NOT cross them:
    //   - Wrapper/top-level: Pkh, Sh, Wpkh, Wsh, Tr, Bare (0x02-0x07)
    //   - Inner-script: SortedMulti, Multi, MultiA, AndV, ..., RawPkH, hash-locks (0x09-0x23)
    //   - Key-slot: Placeholder (0x32), and future Reserved* (0x24-0x31) when subsystem 5 lands
    // The Sh -> {any inner-script tag} REJECTED row in the matrix is correct
    // ONLY because key-slot-family tags never appear directly under Sh.
    // Subsystem 5 implementers must NOT widen this dispatch to admit Reserved* tags.
    let inner_byte = cur.peek_byte()?;  // peek_byte at cursor.rs:111-119
    match Tag::from_byte(inner_byte) {
        Some(Tag::Wpkh) => {
            cur.read_byte()?;
            let key = decode_placeholder(cur, keys)?;
            Ok(Descriptor::new_sh_wpkh(key)?)
        }
        Some(Tag::Wsh) => {
            cur.read_byte()?;
            let wsh = decode_wsh_body(cur, keys)?;
            Ok(Descriptor::new_sh_with_wsh(wsh))  // takes Wsh<Pk>, infallible
        }
        Some(other) => Err(Error::PolicyScopeViolation(format!(
            "v0.4 does not support sh({other:?}); only sh(wpkh(...)) and sh(wsh(...)) allowed"
        ))),
        None => Err(Error::InvalidBytecode {
            offset: cur.offset(),
            kind: BytecodeErrorKind::UnknownTag(inner_byte),
        }),
    }
}
```

fn default_indicator_for_v0_4_types(d: &Descriptor<DescriptorPublicKey>) -> Option<u8> {
    use miniscript::descriptor::{Descriptor, ShInner};
    match d {
        Descriptor::Wpkh(_) => Some(0x03),                  // BIP 84

        _ => None,                                           // unreachable in v0.4
